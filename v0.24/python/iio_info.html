iio_info¶

iio_info is part of the Libiio package, a library that has been developed to ease the development of software interfacing Linux Industrial I/O (IIO) devices.
This tool is written using the libiio Python bindings. It works in the same way as the base iio_info works. You can find more informations about it on this page.
